# Career Plan App
Mobile App project for tracking and organization of career plans made with Xamarin 5.

# NuGet & Techonologies
* **Xamarin.Forms 5**
* Xamarin.Essentials `1.6.1`
* Syncfusion.Xamarin.SfProgressBar `19.3.0.57`
* Newtonsoft.Json `13.0.1`
